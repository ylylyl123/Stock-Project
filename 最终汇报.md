# 多因子量化策略回测报告

## 📊 策略概述

### 策略名称
**优化5因子多维度选股策略**

### 策略逻辑
本策略基于量化金融理论，从**动量、价值、质量**三个维度构建多因子选股模型，旨在捕捉A股市场的超额收益。

### 因子组合

| 因子名称 | 类型 | 权重 | 经济学逻辑 |
|---------|------|------|-----------|
| **动量因子（20日）** | 趋势 | 30% | 趋势延续效应：价格动量反映市场资金流向 |
| **反转因子（5日）** | 反转 | 15% | 短期超跌反弹：捕捉过度反应带来的回归机会 |
| **EP估值因子** | 价值 | 25% | 价值投资核心：低估值股票具有均值回归特性 |
| **BP市净率** | 价值 | 15% | 补充估值维度，筛选账面价值被低估的公司 |
| **成交量异常** | 质量 | 15% | 资金关注度指标：异常放量反映潜在投资机会 |

### 数据来源
- **数据库**：聚源JYDB数据库
- **股票池**：A股全市场
- **数据频率**：日频行情数据

1. **风险控制**：加入止损机制，控制最大回撤

3. **动态调整**：根据市场状态动态调整因子权重
4. **扩展因子**：纳入流动性、波动率等额外维度

## 附录：因子计算公式

### 1. 动量因子（20日）
```
Momentum = (Price_t - Price_{t-20}) / Price_{t-20} × 100
```

### 2. 短期反转（5日）
```
Reversal = -(Price_t - Price_{t-5}) / Price_{t-5} × 100
```

### 3. EP估值因子
```
EP = 1 / Price × 1000
```

### 4. BP市净率代理
```
BP = MA_250(Price) / Price
```

### 5. 成交量异常
```
VolumeAnomaly = Volume_t / MA_20(Volume)
```

### 综合得分
```
Score = 0.30×Momentum + 0.15×Reversal + 0.25×EP + 0.15×BP + 0.15×VolumeAnomaly
```
